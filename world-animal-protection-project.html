How I made this website
The first thing that did was tring to find a CMS to use and during the researches I found out that there are so many different kinds of CMS platforms, out of those CMS platforms that I found, Wordpress is the best suitable CMS to what I am about to develope.
This image is me trying to setup some customizers so that the user or site visitors are able to use the wordpress customizer to change their content and also the second image is me seting up some things at the dashboard of the wordpress. After all the set up and changeing the things in the dashboard, the next thing to do is to do the testings!
For my site most of the thing including the most important things are all created using the wordpress's backend dashboard. First if a user would like to create an article of an animal they have to go to the back end of the wordpress and using the dashboard's creating post function to create a post/articles and if the user would like to manage the about us page they also have to go to the dashboard to do so, same is managing a product. After I finished the project and did a 1 hour training with the client, the client didn't find the wordpress dashboard really hard to use, they actually pick it up really fast.
